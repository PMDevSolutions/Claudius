v1.2.0 (2026-05-26) introduced versioned CDN distribution. No breaking
changes — existing self-hosted embeds keep working untouched.

## Recommended: switch to the CDN channel

Replace your self-hosted bundle references with the version-pinned jsDelivr
channel to get patches and minors automatically:

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/gh/PMDevSolutions/Claudius@1/cdn/claudius.css"
/>
<script src="https://cdn.jsdelivr.net/gh/PMDevSolutions/Claudius@1/cdn/claudius.iife.js"></script>
```

If you do:

- Allow `cdn.jsdelivr.net` in your CSP (`script-src`, `style-src`)
- `@1` never serves a future `2.x` build, so majors stay opt-in

Details: [Embed via CDN](/deployment/cdn/). v1.2.1 was a CI-only patch.
